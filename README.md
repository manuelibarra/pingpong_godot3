# pingpong_godot3
This is a simple PingPong game, it is an initial game written in Godot V3, and you can improve additional characteristics
Godot stable Version  3.5.2
Some items applied are:
-Node 2D
-Collision Shape 2D
-Kinematic Body 2D
-Physics process
-Rigid Body 2D
